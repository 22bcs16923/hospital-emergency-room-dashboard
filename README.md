# 🏥 Hospital Emergency Room Dashboard

## 📊 Overview
Developed an interactive dashboard to analyze hospital emergency room operations and improve patient flow efficiency.

## 🛠 Tools & Technologies
- Excel
- Power Query
- Power Pivot
- DAX

## 📈 Key Metrics
- Total Patient Count
- Average Waiting Time
- Patient Satisfaction Score
- Admission Trends
- Gender & Age Distribution

## 🚀 Features
- Interactive slicers (Month/Year)
- Dynamic KPI tracking
- Department-wise performance insights
